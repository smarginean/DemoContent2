Créer Une Nouvelle Intégration D'Application
Plate-Forme
Vous pouvez joindre une application mobile native à l'application Web SWA ou SAML.
Méthode de connexion
Authentification Web Sécurisée (Swa)
Informations d'identification de l'utilisateur pour se connecter. Cette intubation fonctionne avec la plupart des applications.
SAML 2.0
Utilise le protocole SAML pour connecter les utilisateurs à l'application. C'est une meilleure option que SWA, si l'application la prend en charge.
Connexion OpenID
Utilise le protocole OpenID Connect pour connecter les utilisateurs à une application que vous avez créée.
